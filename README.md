# Legends of Wpierdol
Turn-based fighting game running in your console!
## Formerly a Batch game - now a recreated in C++
### *Fight in a way better than before!*
- C++ Version of LoW is faster
- NOW it has arrow choice selection! No more typing numbers!
### *New content is easier to make!*
- I won't have to worry about some game breaking bug which I would have to try to fix for months!
### *Become a Legend!*
- Encounter harder enemies as the game goes on
- Get new weapons
- WPIERDOL your enemies to show who is the boss
